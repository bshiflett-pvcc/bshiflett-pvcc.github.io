Operating Systems Terminology

Term Definition
operating System software that controls the overall operation of a computer
job queue a list of different tasks that are waiting to be done by the operating system
FIFO Stands for the phrase "first-in first-out" which basically means that the first item that gets added to a queue will also be the first item to be removed and proccessed.
interactive processing a program that carries on a dailogue with the user while being excecuted
time-sharing lets multiple users access the computer and its programs at the same time
multiprogramming technique where a excution of each job is restricted to one intervalat a time
application software a specific type of computer software that only does certain tasks for the user
system software consists of the programs for performing tasks particular to the machines utilization
utility software programs for performing activities that are fundamental to computer installations but not included in the operating system
graphical user interface (GUI) preforms actions requested by the computers users and handles the communication
shell these are older user interfaces
kernel internal part of a operating system
device driver oftware units that communicate with the controllers
file manager coordinates the use of the machines mass storage facilities
memory manager charged with the task of coordinating the machines use of main memory.
virtual memory memory managements way that allows programs to execute which makes them think that they have more memory then they actually do
boot strapping t is this procedure that transfers the operating system from mass storage (where it is permanently stored) into main memory
ROM contains a collection of software routines for performing fundamental input/output activities such as receiving information from the keyboard, displaying messages on the computer screen, and reading data from mass storage
deadlock condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another
spooling technique of holding data for output at a later but more convenient time
super user (or administrator) a priviledged account with access to all settings on the computer
multi-factor authentication security method that makes users provide more than one authentification to access the system
