Network and Internet Terminology

  • Network classifications:
  • LAN :a collection of devices connected together in one physical location, such as a building, office, or home
  • MAN:a computer network that connects computers within a metropolitan area
  • WAN: a large computer network that connects groups of computers over large distances
  • Network Models:
  • bus network: local area network (LAN) topology in which each node -- a workstation or other device -- is connected to a main cable or link called a bus
  • star network: a local area network (LAN) topology in which all nodes -- personal computers (PCs), workstations or other devices -- are directly connected to a common central computer that is often referred to as a hub.
  • Protocols:
  • CSMA/CD: a medium access control (MAC) method used most notably in early Ethernet technology for local area networking
  • CSMA/CA: a protocol for carrier transmission in 802.11 networks
  • Combining Networks:
  • Repeater: a powerful network hardware device that regenerates an incoming signal from the sender before retransmitting it to the receiver
  • Bridge: a computer networking device that creates a single, aggregate network from multiple communication networks or network segments
  • Switch: equipment that allows two or more IT devices, such as computers, to communicate with one another
  • Router: a device that connects two or more packet-switched networks or subnetworks
  • Gateway: a device or node that connects disparate networks by translating communications from one protocol to another.
  • Client/ Server Model:
  • P2P: a group of computers are linked together with equal permissions and responsibilities for processing data
  • Distributed Sytem: any environment where multiple computers or devices are working on a variety of tasks and components, all spread across a network.
  • Cloud Computing: toring and accessing data and programs over the internet instead of your computer's hard drive
  • Internet Addrtessing:
  • Tier 1: an Internet Protocol (IP) network that can reach every other network on the Internet solely via settlement-free interconnection
  • Tier 2: a service provider that utilizes a combination of paid transit via Tier 1 ISPs and peering with other Tier 2 ISPs to deliver Internet traffic to end customers through Tier 3 ISPs
  • Tier 3: a provider that strictly purchases Internet transit
  • IP Address:
  • ICANN:
  • DNS:turns domain names into IP addresses, which browsers use to load internet pages
  • TLD: the last segment of a domain name, or the part that follows immediately after the "dot" symbol.
  • Country-Code TLDS:
  • Internet Protocolos:
  • FTP: a standard network protocol used for the transfer of files from one host to another over a TCP-based network, such as the Internet
  • HTTP: an application protocol for distributed, collaborative, hypermedia information systems that allows users to communicate data on the World Wide Web.
  • SMTP: A messaging protocol governing electronic-mail transmission in Transmission Control Protocol/Internet Protocol (TCP/IP) networks
  • World Wide Web:
  • URL: the mechanism used by browsers to retrieve any published resource on the web
  • client side-activities:everything in a web application that is displayed or takes place on the client (end user device)
  • server-side activities: activites that are carried out in response to client requests, and non-client-oriented operations such as maintenance tasks
  • Internet Four-Layer Software:
  • Application Layer: the layer for collecting and managing the data that gets produced.
  • Transport Layer:provides transparent transfer of data between end users, providing reliable data transfer services to the upper layers
  • Network Layer: the part of the Internet communications process where these connections occur, by sending packets of data back and forth between different networks
  • Link Layer: includes the protocols that define communication between local (on-link) network nodes which fulfill the purpose of maintaining link states between the local nodes
  • Internet is:
  • Packet-Switching Network: transfers small pieces of data across various networks.
  • Fault Tolerant:a process that enables an operating system to respond to a failure in hardware or software.
  • Redundant: the process of providing multiple paths for traffic so that data can keep flowing even in the event of a failure.
  • Cybersecurity
  • Malware: software that is specifically designed to disrupt, damage, or gain unauthorized access to a computer system.
  • Virus: a piece of code that is capable of copying itself and typically has a detrimental effect, such as corrupting the system or destroying data.
  • Worm: a type of malware or malicious software that can replicate rapidly and spread across devices within a network
  • Trojan Horse: a type of malware that downloads onto a computer disguised as a legitimate program
  • Spyware: malicious software that enters a user's computer, gathers data from the device and user, and sends it to third parties without their consent
  • Phishing : the practice of sending fraudulent communications that appear to come from a legitimate and reputable source, usually through email and text messaging
  • DDos Attak: designed to force a website, computer, or online service offline
  • ProtectionL:
  • Firewall: a network security device that monitors incoming and outgoing network traffic and permits or blocks data packets based on a set of security rules.
  • Spam filter: a program used to detect unsolicited, unwanted and virus-infected emails and prevent those messages from getting to a user's inbox.
  • Proxy Server: a system or router that provides a gateway between users and the internet.
  • Antivirus Software: A program that monitors a computer or network to identify all major types of malware and prevent or contain malware incidents.
  • Cryptography: the process of hiding or coding information so that only the person a message was intended for can read it
  • Public Key Encryption: a large numerical value that is used to encrypt data.
