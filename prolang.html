Chapter 6: Programming Languages

TERM Definition
Machine Language Binary code used by computer processor
Debugging identifies and removes all errors in your program
Assembler a type of computer program that takes in basic instructions and converts them into a pattern of bits that the computer's processor can use to perform basic operations.
Assembly Language a type of low-level programming language that is intended to communicate directly with a computer's hardware.
Translator the process of converting code written in one programming language to another programming language while maintaining the same functionality
compiler a special program that translates a programming language's source code into machine code, bytecode or another programming language.
interpreter a program that directly executes the instructions in a high-level language, without converting it into machine code
cross-platform software a type of application / program / software that works on various operating systems or devices
natural language he ability of a computer program to understand human language as it is spoken and written
formal language a set of rules that define how code should be written
machine independence where the program can be used on different hardware and software configurations
high-level programming language allows humans to work with computers by writing code that is closer to a natural language
object-oriented programming (OOP) a computer programming model that organizes software design around data, or objects, rather than functions and logic
scripting language used to manipulate, customize, and automate the facilities of an existing system
integer (Data Type) an optimisation method that involves the use of integer variables, or “whole numbers”
float (Data Type) a variable that contains decimal numbers
string (Data Type) a sequence of characters used to represent text.
primitive data type one of a set of fundamental language elements that serve as the foundation for a programming language
data structure way to organize your data in a program
python collection data structures (pycds)
list (pycds) a sequence of several variables, grouped together under a single name
tuple (pycds) ordered lists with a finite number of elements
set (pycds) a data structure that stores unique elements of the same type in a sorted order
dictionary (pycds) a key-value store
function an approach to software development that uses pure functions to create maintainable software.
local variable a variable that is declared outside any function and is accessible to all routines in our program
global variable a variable with global scope, meaning that it is visible (hence accessible) throughout the program, unless shadowed
source program programming statements that are created by a programmer with a text editor or a visual programming tool and then saved in a file
object program a programming paradigm (and not a specific language) built on the concept of objects
